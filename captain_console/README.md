# Captain Console

## Environmental Variables
Eftirfarandi env breytur þurfa að vera rétt stilltar til að ná gagnagrunnstengingu.
- DB_NAME
- DB_USER
- DB_PASSWORD
- DB_HOST
- DB_PORT
- PORT=8000 # Svo heroku local sé á réttu porti.
